## 三、标准煤折算系数

> **重要**: 电力折标煤有两种方法，中国建筑能耗统计和标准对标使用 **等价值**。

### 电力

| 折算方法 | 系数 | 单位 | 说明 | 适用场景 |
|---------|------|------|------|---------|
| **等价值**（推荐） | 0.298 | kgce/kWh | 基于全国平均供电煤耗率 | 建筑能耗统计、国标对标、能源审计 |
| 当量值 | 0.1229 | kgce/kWh | 1 kWh = 3.6 MJ 热值等效 | 国际标准(LEED/BREEAM)、碳排放计算边界 |

> 注：等价值根据全国火电厂平均供电煤耗每年更新（近年约 295-305 gce/kWh）。
> 本技能默认使用 0.298 kgce/kWh 等价值，用户可指定自定义系数。

## 四、常用计算公式

### 电力碳排放
```
CO₂_电 = 用电量(kWh) × 区域电网排放因子(kgCO₂/kWh) / 1000
```
结果单位：tCO₂

### 天然气碳排放
```
CO₂_气 = 用气量(m³) × 1.997(kgCO₂/m³) / 1000
```
结果单位：tCO₂

### 市政热力碳排放
```
CO₂_热 = 用热量(GJ) × 热源排放因子(tCO₂/GJ)
```
结果单位：tCO₂

### 总碳排放
```
CO₂_总 = CO₂_电 + CO₂_气 + CO₂_热
```

### 折标准煤
```
kgce = 用电量(kWh) × 0.1229 + 用气量(m³) × 1.2143 + 用热量(GJ) × 34.12
```
