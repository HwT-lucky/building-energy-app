# 建筑节能与碳排放标准

> 数据来源：各标准正式文本。表格数据均为标准原文引用。

## 一、DB31/T 783-2026《高等学校建筑合理用能指南》

> 上海市地方标准，2026-05-01发布，2026-08-01实施，代替DB31/T 783-2014

### 适用范围
上海市本科院校、高职（专科）院校、成人高等学校在日常运行中各类建筑能源消耗的计算和评价。

### 表1：单位建筑面积年综合能耗指标 (e1) [kgce/(m²·a)]

| 学校类别 | 约束值 | 基准值 | 引导值 |
|---------|--------|--------|--------|
| 本科院校/成人高等学校 | 36.3 | 25.9 | 19.3 |
| 高职（专科）院校 | 24.6 | 18.2 | 15.9 |

### 表2：生均年综合能耗指标 (e2) [kgce/(p·a)]

| 学校类别 | 约束值 | 基准值 | 引导值 |
|---------|--------|--------|--------|
| 本科院校/成人高等学校 | 800 | 410 | 350 |
| 高职（专科）院校 | 380 | 270 | 225 |

### 修正系数

#### 表3：单位建筑面积教学科研设备资产修正系数 (α1)

| 设备资产 Aua (元/m²) | α1 |
|---------------------|-----|
| <500 | 0.9 |
| [500, 1000) | 1.0 |
| [1000, 2000) | 1.1 |
| [2000, 3000) | 1.2 |
| ≥3000 | 1.25 |

#### 表4：一流学科门类修正系数 (α2)

| 学科门类 | α2 |
|---------|-----|
| 艺术学 | 1.05 |
| 文学（含哲学、法学、历史学、管理学） | 1.08 |
| 经济学、教育学 | 1.1 |
| 理学、工学 | 1.3 |
| **医学、农学** | **2.0** |
| 三种一流学科门类及以上 | 1.7 |

#### 表6：硕博研究生占比修正系数 (α4)

| 硕博占比 | α4 |
|---------|-----|
| <30% | 1.0 |
| [30%, 40%) | 1.5 |
| [40%, 50%) | 2.0 |
| ≥50% | 2.5 |

### 计算公式

本科院校：
- e1 = E / (S × α1 × α2)
- e2 = E / (R × α2 × α4)

高职院校：
- e1 = E / (S × α1 × α3)
- e2 = E / (R × α3)

### 对标规则

1. e1 ≤ 引导值 → 领先水平，持续保持
2. 引导值 < e1 ≤ 基准值 → 宜采取更有效节能措施，达到引导值
3. 基准值 < e1 ≤ 约束值 → 设定年度提升目标，逐步达到基准值
4. e1 > 约束值 → 需比对生均年综合能耗(e2)，满足表2约束值
