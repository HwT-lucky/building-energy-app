# 计算方法说明

## 一、能源比例分析

### 步骤

1. 将各类能源消耗折算为标准煤 (kgce)：
   - 电力: `标煤量 = 用电量(kWh) × 0.1229`
   - 天然气: `标煤量 = 用气量(m³) × 1.2143`
   - 热力: `标煤量 = 用热量(GJ) × 34.12`

2. 计算总标煤量：
   ```
   总标煤量 = Σ 各类能源标煤量
   ```

3. 计算各能源占比：
   ```
   占比 = 该类能源标煤量 / 总标煤量 × 100%
   ```

### 输出示例
```
| 能源类型 | 实物量 | 折标煤 (kgce) | 占比 |
|---------|--------|-------------|------|
| 电力     | 500,000 kWh | 61,450 | 55.2% |
| 天然气   | 30,000 m³   | 36,429 | 32.7% |
| 热力     | 400 GJ      | 13,648 | 12.1% |
| 合计     | —           | 111,527 | 100% |
```

## 二、逐月用能量分析

### 指标

1. **逐月总能耗趋势**: 每月各类能源折标煤之和
2. **分项能耗趋势**: 电力、天然气、热力逐月变化
3. **峰值月**: 能耗最高的月份
4. **谷值月**: 能耗最低的月份
5. **季节性特征**: 夏季制冷高峰 vs 冬季采暖高峰
6. **峰谷比**: 峰值月能耗 / 谷值月能耗

### 异常检测

- 某月能耗超过月均值的 150% → 标记为"高能耗月"
- 某月能耗低于月均值的 50% → 标记为"低能耗月"，建议核实数据

## 三、碳排放计算

### 总碳排放

```
CO₂总 = CO₂电 + CO₂气 + CO₂热

CO₂电 = 年用电量(kWh) × 区域电网因子(kgCO₂/kWh) / 1000
CO₂气 = 年用气量(m³) × 1.997 / 1000
CO₂热 = 年用热量(GJ) × 热源排放因子(tCO₂/GJ)
```

### 单位面积碳排放强度

```
碳强度 = CO₂总(tCO₂) × 1000 / 建筑面积(m²)
```
单位：kgCO₂/m²·a

## 四、标准对标

### 判定流程

1. 根据建筑所在地，查询 `references/policy_standards.md` 中的地方标准
2. 如无地方标准，使用国家标准
3. 根据建筑类型（办公/商业/酒店/医院/学校等），查询对应的能耗限额
4. 将建筑实际单位面积能耗 (kgce/m²·a) 与限额比较

### 对标等级

| 等级 | 条件 | 标识 |
|------|------|------|
| 优良 | 实际能耗 ≤ 引导值 | 🟢 绿色 |
| 达标 | 引导值 < 实际能耗 ≤ 约束值 | 🟡 黄色 |
| 超标 | 实际能耗 > 约束值 | 🔴 红色 |
| 无标准 | 无匹配的限额标准 | ⚪ 灰色（仅展示计算值） |

## 五、同比分析（可选）

如有往年同期数据：

```
同比增长率 = (今年能耗 - 去年能耗) / 去年能耗 × 100%
```

- 正值表示能耗增加，需关注
- 负值表示节能效果，可总结节能措施
